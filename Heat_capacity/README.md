\# Note on the MoLFormer model files



The pretrained `MoLFormer-XL-both-10pct` model folder is not included in this repository .



Please download the model files separately from the official resource and place the `MoLFormer-XL-both-10pct` folder in this directory before running `Feature\_extraction.py`.



MoLFormer GitHub repository:  

https://github.com/IBM/molformer



Pretrained model page:  

https://huggingface.co/ibm-research/MoLFormer-XL-both-10pct
